# Formato do GLOSSARY.md (workspace de ensino)

Linguagem canônica do workspace de ensino — distinta do `CONTEXT.md` do produto, salvo se forem o mesmo esforço e o humano unificar.

## Estrutura

```md
# Glossário: {Tópico}

{Uma ou duas frases sobre o tópico.}

## Termos

**Termo**:
Definição curta do que o termo é.
_Avoid_: Sinônimo frouxo, gíria
```

## Regras

- Só promova termo quando a pessoa já o usa corretamente.
- Seja opinativo; `_Avoid_` para aliases.
- Definições de 1–2 frases.
- Use termos do próprio glossário dentro de outras definições.
- Agrupe sob subtítulos se ajudar.
- Revise definições conforme o entendimento aprofunda.
