# Formato do MISSION.md

`MISSION.md` fica na raiz do workspace de ensino. Toda decisão de ensino deve traçar até aqui.

## Template

```md
# Missão: {Tópico}

## Por quê
{1–3 frases. Objetivo concreto no mundo real.}

## Sucesso parece
- {Coisa observável que a pessoa conseguirá fazer}
- {…}

## Restrições
- {Tempo, orçamento, preferências…}

## Fora de escopo
- {Tópicos adjacentes que não perseguir agora}
```

## Regras

- Uma missão por workspace.
- Concreto > abstrato.
- Empurre vaguidão com perguntas antes de escrever.
- Revise quando a realidade mudar.
- Curto: se passar de uma tela, virou plano, não bússola.
