# Formato do RESOURCES.md

Fontes confiáveis do tópico. Conhecimento das lições vem daqui, não de chute paramétrico.

## Estrutura

```md
# Recursos: {Tópico}

## Conhecimento

- [Livro: _Título_ — Autor](https://…)
  O que cobre. Use para: …

## Sabedoria (comunidades)

- [Fórum X](https://…)
  Por que confiar. Use para: …
```

## Regras

- Só alta confiança (primárias, especialistas, peer-review, comunidades bem moderadas).
- Anote cada entrada (o quê + quando usar).
- Separe Conhecimento / Sabedoria.
- Liste lacunas em `## Lacunas` se faltar fonte.
- Pode com rigor.
- Se o humano optou por não entrar em comunidades, registre isso.
